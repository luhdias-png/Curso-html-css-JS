Paragrafo e quebra de linhas

Aqui para voce colocar uma linha voce pode usar <hr> que sera criaado uma linha.

Voce pode escrever um paragrafo de qualquer jeito basta colocar tudo no meio de <p> Aqui voce teria que escrever seu paragrafo e fechar com </p>

Se vc quiser quebrar o texto é so usar a tag <br>.
olha como ele quebrou essa linha.

Vamos adicionar alguns simbolos especiais:
® esse codigo é usado com: & reg;
© esse daqui é: & copy;
™ Esse é: &trade ;
Obs. Eles devem estar sempre juntos tanto o ® como o ©.
Existem uma variedade de icones para se colocar no html porem todos devem ter:" &...;"
